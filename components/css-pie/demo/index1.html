CSSpie.js



Pure CSS3 Pie Chart Generator

What is it?

CSSpie.js is JavaScript library that allows you to create a dynamic CSS3 Pie Chart with only 1 line of code.

Try the Generator

This is a the Pie Chart Generator, set the colors and the percentages of each slice and witness the magic.

How it works

Step 1

Include CSSpie.js in your web page.
<script src="CSSpie.js"></script>

Step 2

Create the pie using the following code.
<script>
var pie = createPie(id, size, backColor, numSlices, percentages, colors);
</script>

  1. id: This is the ID of the pie.
  2. size: This is the Size of the pie, pixel values are used here, for example, 200px.
  3. backColor: This is the Background Color or Base Color of the pie. Any type of color variables can be used, HEX, RGB, or color names.
  4. numSlices: This is the Number of Slices in the pie.
  5. percentages: This is an Array of Percentages of the slices.
  6. colors: This is an Array of Colors of the slices.

Step 3

Now we must add the pie to the DOM.
<script>
var pie = createPie(id, size, backColor, numSlices, percentages, colors);
document.getElementById("myDiv").appendChild(pie);
</script>

Examples

createPie("workers","200px","white",5,[10,45,22,11,12],["#FF00FF","#336699","#66CC00","#CC6633","#FFFF99"]);


createPie("students","300px","white",3,[55,12,33],["#CCFFFF","#CCFF66","#003333"]);
